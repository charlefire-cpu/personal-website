# Hero background

Replace `hero-background.jpg` in this folder to update the site hero image.

**Recommended specs**
- Format: JPG or WebP
- Width: 1920px or larger
- Subject: modern workspace, business environment, or subtle corporate texture
- Keep the filename `hero-background.jpg`, or update `HERO_BACKGROUND_IMAGE` in `src/data/profile.ts`

The path is configured in `src/data/profile.ts` as `HERO_BACKGROUND_IMAGE`.
